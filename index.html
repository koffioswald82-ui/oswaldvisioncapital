Note sur les marchés africains, Données manuellesNote on African markets, Manual data

Les bourses africaines (BRVM, JSE, NSE) ne sont pas disponibles sur Yahoo Finance ou Bloomberg gratuit. Les données sont renseignées manuellement depuis les sources officielles. C'est un avantage compétitif :peu d'analystes les couvrent, ce qui crée des opportunités de sous-évaluation. African stock exchanges (BRVM, JSE, NSE) are not available on Yahoo Finance or free Bloomberg. Data is entered manually from official sources. This is a competitive advantage:few analysts cover them, which creates undervaluation opportunities.
